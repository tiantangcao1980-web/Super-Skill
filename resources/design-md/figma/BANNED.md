# Figma — BANNED Patterns

## Typography
- NEVER use serif fonts — Figma uses Whyte / custom sans
- NEVER use dark formal typography — Figma is playful and friendly
- NEVER use more than 3 font weights per page

## Color
- NEVER use muted monochrome — Figma uses bright playful multi-color
- NEVER limit to 1-2 colors — Figma's signature is the 4-color brand palette (red/blue/green/purple)
- NEVER use dark mode on marketing
- NEVER use gradients as background — Figma uses solid colored blocks

## Layout
- NEVER use minimalist hero — Figma shows product UI, customer artwork, or collaboration in action
- NEVER hide the colorful shapes that are Figma's signature decoration
- NEVER use 3-column feature grids — Figma uses interactive demos inline

## Components
- NEVER use gray boring buttons — Figma CTAs are vibrant colored
- NEVER use sharp 0px borders — Figma prefers `border-radius: 8-12px`
- NEVER hide collaboration indicators (multiplayer cursors, avatars) in product screenshots

## Motion
- NEVER avoid motion — Figma's site has constant playful movement
- NEVER use harsh/short transitions — Figma uses 300-500ms gentle motion
- NEVER skip the "components animating into place" moments

## Copy
- NEVER use formal corporate voice — Figma is conversational, often witty
- NEVER use "enterprise" as the hero framing — Figma leads with craft
- NEVER use stat-heavy hero — Figma leads with imagery

## Signature "don'ts"
- NEVER show Figma the tool without showing the design artifacts IN it
- NEVER omit collaboration signals (multiple cursors, comments, avatars)
- NEVER use black as the brand color — Figma is color-rich
