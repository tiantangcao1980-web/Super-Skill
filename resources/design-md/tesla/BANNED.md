# Tesla — BANNED Patterns

## Typography
- NEVER use serif fonts anywhere on the marketing site
- NEVER use more than 2 font weights per page (Regular + Medium)
- NEVER use decorative display fonts — Universal Sans or Gotham-like only
- NEVER use justified text alignment

## Color
- NEVER use warm colors as primary — Tesla's palette is white/black/gray with rare red accent
- NEVER use gradients (absolutely none)
- NEVER use colored text except for error states
- NEVER color icons or illustrations

## Layout
- NEVER use more than 3 screens of content on a product page
- NEVER break the full-viewport photography format
- NEVER place text over the center of product photography — always margins
- NEVER use sidebars, multi-column grids, or content-heavy layouts
- NEVER show a "grid of features" — Tesla shows one product at a time

## Components
- NEVER use cards with visible borders
- NEVER use shadows (flat or photographic backgrounds only)
- NEVER use rounded corners larger than 4px
- NEVER use pill buttons — Tesla uses near-square CTAs
- NEVER use animated SVG icons

## Motion
- NEVER animate the product photography — cinematic still frames only
- NEVER use parallax scrolling on the body; optionally sticky CTA only
- NEVER use fade-in-on-scroll reveals — content appears instantly

## Copy
- NEVER use more than 8 words in a headline
- NEVER use jargon or technical specs in the hero — specs go in tables below
- NEVER use "AI-powered," "smart," or buzzwords
- NEVER use emoji, ever

## Signature "don'ts"
- NEVER show the product interior from the exterior shot — sections are discrete
- NEVER mix marketing copy with configurator — one or the other per page
- NEVER use a navigation mega-menu — Tesla uses hamburger or top-bar text links only
