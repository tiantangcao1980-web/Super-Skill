# Notion — BANNED Patterns

## Typography
- NEVER use techno or futuristic sans serifs — Notion uses a warm humanist sans (Inter / SF-like)
- NEVER use more than 3 font weights
- NEVER use display fonts at small sizes (serif displays are large-only)
- NEVER use italic except for block quotes

## Color
- NEVER use saturated or neon colors — Notion's palette is warm neutrals with soft accents
- NEVER use pure `#000` — warm near-black `#37352f` is the signature text color
- NEVER use pure `#fff` for the page surface — `#ffffff` with warm overlays or `#f7f6f3` off-white
- NEVER use cool grays — Notion's grays are warm with yellow/brown undertone

## Layout
- NEVER use dense dashboards — Notion's marketing leans editorial, not data
- NEVER use dark mode on marketing (product has dark mode; landing does not)
- NEVER use sticky CTAs that follow scroll
- NEVER use full-bleed imagery over 60vh

## Components
- NEVER use neon gradient buttons — Notion CTAs are solid warm black or white
- NEVER use animated feature cards — Notion shows screenshots with subtle float
- NEVER use avatar stacks for "team of X"
- NEVER use high-contrast borders — Notion borders are `#e9e9e7` soft

## Motion
- NEVER use bouncy springs or heavy parallax
- NEVER animate icons or illustrations continuously
- NEVER use scroll-jacking

## Copy
- NEVER use "AI-powered" or "workflow automation" as a primary claim — lead with user benefit
- NEVER use hype words ("revolutionary," "game-changing")
- NEVER use exclamation marks
- Headlines are conversational and human: "All you need, in one place" not "Industry-leading productivity suite"

## Signature "don'ts"
- NEVER hide the hand-drawn illustrations — they're central to Notion's warmth
- NEVER show the product UI without the warm cream canvas behind it
- NEVER use enterprise logos bar — Notion uses customer quotes with real names and real products
