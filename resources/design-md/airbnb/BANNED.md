# Airbnb — BANNED Patterns

## Typography
- NEVER use techno or geometric sans — Airbnb uses its Cereal font (rounded humanist)
- NEVER use ultra-thin weights (<400) at body sizes
- NEVER use condensed or expanded widths

## Color
- NEVER use cold blues or tech gray as primary — Airbnb is warm coral `#FF385C` on cream
- NEVER use pure black as accent — soft warm `#222` for text
- NEVER use monochrome marketing — Airbnb imagery is lush and photographic

## Layout
- NEVER omit photography — every page has real user-generated imagery
- NEVER use wireframes or placeholder boxes instead of photos
- NEVER center all content — Airbnb uses 3-column photography grids with heavy imagery
- NEVER use full-dark mode on marketing

## Components
- NEVER use sharp corners — rounded corners everywhere (`border-radius: 12px` or pill)
- NEVER use hard shadows — soft, large-radius shadows with warm tint
- NEVER use skeleton product UI in screenshots — show real listings

## Motion
- NEVER use snappy/short transitions — Airbnb motion is 400-500ms warm ease
- NEVER use harsh spring bounces
- NEVER auto-play videos without sound warning

## Copy
- NEVER use corporate voice — Airbnb speaks in human, second-person stories
- NEVER use "user" as a noun — Airbnb says "guest" and "host"
- NEVER use emoji in body copy (only in reviews UI)
- NEVER use stock metrics

## Signature "don'ts"
- NEVER use generic stock imagery — Airbnb uses real host photography
- NEVER use "best in class" style hype
- NEVER style search as an enterprise dropdown — Airbnb search is the iconic pill
