# Claude (Anthropic) — BANNED Patterns

## Typography
- NEVER use tech-aggressive fonts — Claude uses a warm humanist sans (Styrene or equivalent)
- NEVER mix more than 2 font families
- NEVER use display weights (800+) for UI
- NEVER use sans-only hierarchy without the serif Tiempos Headline for editorial moments

## Color
- NEVER use AI-cliché purples (`#8B5CF6` family) — Claude uses terracotta `#C96442` / warm amber
- NEVER use pure black or pure white — Claude's canvas is `#faf9f7` cream, text is `#3d3929`
- NEVER use cool grays — everything warms toward brown/amber
- NEVER use rainbow gradients or mesh gradients

## Layout
- NEVER use densely packed feature grids — Claude breathes with generous whitespace (min `py-24`)
- NEVER use aggressive hero imagery — Claude leans editorial-long-form
- NEVER center body text — always left-aligned with 65ch max-width
- NEVER use full-bleed video auto-playing in hero

## Components
- NEVER use sharp-edged cards — Claude uses `border-radius: 16px` and up
- NEVER use colored shadows — warm neutral shadows only
- NEVER use "robotic" iconography — Claude uses hand-adjusted line icons with personality
- NEVER use hard dividers — whitespace separates

## Motion
- NEVER use jittery or glitchy effects
- NEVER use aggressive spring bounces
- NEVER auto-advance carousels
- Motion is gentle: 400ms ease-out default

## Copy
- NEVER use corporate AI language ("AI-powered," "harness the power of AI")
- NEVER claim superhuman capabilities — Claude's voice is humble and precise
- NEVER use emoji except in chat UI demonstrations
- NEVER use fake metrics or "10x" language

## Signature "don'ts"
- NEVER style Claude as a "futuristic AI assistant" with neon/chrome/holographic aesthetics — Claude is warm-editorial, literary
- NEVER show generic chat-bubble product screenshots with "I'm an AI assistant" copy
- NEVER use stock AI imagery (brain circuits, glowing networks, holograms)
