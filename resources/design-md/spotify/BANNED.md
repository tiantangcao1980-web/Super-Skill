# Spotify — BANNED Patterns

## Typography
- NEVER use serif fonts — Spotify is pure sans (Circular / Spotify Mix)
- NEVER use thin weights at body size
- NEVER use uppercase at body size — only in small labels/eyebrows
- NEVER mix more than 2 families

## Color
- NEVER use white backgrounds on marketing — Spotify is near-black `#121212` primary
- NEVER use cool blues — Spotify's palette is Spotify Green `#1ED760` on black/purple/pink
- NEVER use more than 2 accent colors per view
- NEVER use pastel palette — Spotify is vibrant saturated

## Layout
- NEVER omit album artwork or artist imagery — visuals drive every page
- NEVER use centered heroes without imagery
- NEVER use dense text grids — Spotify is image-first, text-second

## Components
- NEVER use square play buttons — Spotify uses circles
- NEVER use outlined iconography — filled or weight-filled only
- NEVER use flat cards — every card has hover lift + color-extract from album art
- NEVER use hard borders — Spotify uses `rgba(255,255,255,0.1)` subtle separators

## Motion
- NEVER skip micro-animations on cards — Spotify has hover play-button reveal
- NEVER use fade-in-on-scroll for music content — it should feel immediate

## Copy
- NEVER use corporate speak — Spotify's voice is casual and energetic
- NEVER use "AI" as a feature name — Spotify says "Made For You," "Discover Weekly"
- NEVER use stats in marketing copy — Spotify shows playlists, not numbers

## Signature "don'ts"
- NEVER show album artwork in monochrome or with heavy filters
- NEVER use "Now Playing" with text-only representation — always include cover art
- NEVER show "Premium" as a dry feature comparison — frame as lifestyle benefits
