# Linear — BANNED Patterns

## Typography
- NEVER use Inter as a body font (Linear uses its own Inter Display tuned variant)
- NEVER use anything bigger than `text-[56px]` for hero headlines
- NEVER use thin (<400) weights at small sizes — Linear's floor is 400
- NEVER use serif anywhere

## Color
- NEVER use warm colors (no oranges, reds, ambers) as primary UI
- NEVER use pure black — Linear's dark mode is `#08090a` with tinted surfaces `#101113`
- NEVER use more than one accent (indigo/purple `#5e6ad2` family)
- NEVER use saturated greens or yellows

## Layout
- NEVER use decorative backgrounds
- NEVER use full-bleed sections with card overlays
- NEVER exceed 1280px content width
- NEVER center long-form text (always left-aligned)
- NEVER use image-first heroes — Linear opens with text and a product screenshot

## Components
- NEVER use soft-shadowed cards — Linear uses 1px borders `#2e2e33`
- NEVER use avatars or profile images in the marketing site
- NEVER show "Pricing comparison tables" longer than 10 rows
- NEVER use animated feature cards — screenshots are static

## Motion
- NEVER use entrance animations on scroll
- NEVER use motion heavier than cursor hover feedback + modal slide
- NEVER use loading spinners; Linear uses shimmer skeletons

## Copy
- NEVER use corporate speak ("solutions," "empowering," "synergy")
- NEVER use exclamation marks
- NEVER address the user with generic "You deserve..." framing
- Headlines are object-first ("Issue tracking for engineers") not verb-first

## Signature "don'ts"
- NEVER use illustrations or marketing graphics — screenshots of the product only
- NEVER use gradient overlays on product screenshots
- NEVER show "How it works" step-by-step videos — Linear's magic is in the interactions, not explainers
