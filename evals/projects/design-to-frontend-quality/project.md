# Design To Frontend Quality

## User Goal

Use DesignDNA and UI framework skills to build a polished product surface with accessibility, responsive layout, and implementation checks.

## Expected Workflow

Brand/design system selection -> UI framework adapter -> frontend patterns -> visual/accessibility QA -> output quality gate.

## Acceptance Evidence

- DesignDNA is available with brand systems and generated artifacts.
- Frontend/design skills are discoverable without duplicate installable names.
- DesignDNA CLI tests and brand synchronization checks pass.
