# Design Quality

design-craft-gate starts the work with PRODUCT.md and DESIGN.md context before UI mutation.

DesignDNA is the source of truth for brand colors, spacing, radius, and component behavior.

The frontend quality gate checks accessibility labels, responsive constraints, stable dimensions, deterministic anti-pattern signals, and a quiet product UI that avoids decorative excess.

Acceptance: the rendered dashboard must expose brand context, use real tokens, avoid generic AI design patterns, and pass deterministic quality checks.
