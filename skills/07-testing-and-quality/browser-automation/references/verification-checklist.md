# Browser Verification Checklist

- Start the app with the documented dev command.
- Open the real URL in a browser.
- Check desktop and mobile widths.
- Exercise the primary workflow, not only page load.
- Capture screenshot evidence for visual work.
- Inspect console errors and failed network requests.
- Convert exploratory steps into Playwright only after the target flow is understood.

Report command, URL, viewport, workflow checked, and remaining gaps.
