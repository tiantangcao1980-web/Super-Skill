# Resource Scenario Map

| Need | Start With |
| --- | --- |
| Create a new skill | `skill-authoring-system` |
| Package for multiple agents | `cross-tool-packaging` |
| Verify before shipping | `verification-loop` |
| Build UI | `designdna`, DesignDNA brand specs, frontend patterns |
| Debug code | `debugging`, `code-review`, `security-review` as needed |
| Ship repository work | `git`, `github`, `deployment-patterns` |

Prefer local repository resources first, then official docs or current upstream sources when facts may have changed.
