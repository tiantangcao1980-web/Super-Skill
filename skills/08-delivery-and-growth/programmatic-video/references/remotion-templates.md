# Remotion Templates

## Simple Composition

```tsx
export const Demo = () => (
  <AbsoluteFill style={{ background: '#0f172a', color: 'white' }}>
    <h1>Product demo</h1>
  </AbsoluteFill>
);
```

## Practical Defaults

- 1920x1080 for launch videos.
- 1080x1080 for social square cuts.
- 30 fps unless motion needs more.
- Store reusable text, colors, and timing in constants.
